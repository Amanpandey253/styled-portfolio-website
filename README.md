# Aman Pandey Portfolio Website

## Project Overview

This project is a responsive personal portfolio website created using HTML5 and CSS3. The website showcases my skills, projects, and contact information in a clean and professional design.

The portfolio was developed as part of Week 2 (CSS - Styling Your Website) and focuses on responsive design, modern layouts, and user-friendly navigation.

## Objectives

- Create a professional personal portfolio website.
- Apply CSS styling to improve visual appearance.
- Practice Flexbox/Grid layouts.
- Implement responsive design for mobile devices.
- Add interactive hover effects and smooth transitions.
- Organize website content using semantic HTML.

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid
- Media Queries
- Git & GitHub
